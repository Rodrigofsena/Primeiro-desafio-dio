Alumas Tegs e Atributos para que elas servem.

  1. Teg <i > </i > :Ela deixa a frase ou nome selecionado em intalico
  2. Teg <strong ></strong >: Deixa a frase ou nome selecionado em negrito.
  3. Teg <input> : Ele cria um campo que o usuario pode digitar algo, podendo mudar de texto para numero e ate cor.
  4. Atributo Type="text", atributo que complementa a teg input, fazendo assim definir o tipo de informação que vai colocar.
  5. Teg <img src="" alt=""> : Usada para adicionar uma imagem ao site.
  6. Atributo id="" = Ele é um indentificador de elementos.
  7. Atributo Style=""= É um atriduto voltado para CSS usado para mudar o estilo daquilo que desejar, seja cor,forma etc .
  8. Atridubuto width="250" = Com ele você pode escolhe largura da imagem que desejar alterar.
  9. Teg <mark > </mark> :usada para marca o texto
  10. A teg <h1> </h1> :ela é o usado para colocar o titulo do texto, essa teg vai ate (h2) sendo elas os subtitulos.
  11. A teg <p> </p> :ela é usada para colocar o paragrafo, onde você vai poder o corpo do texto.
  12. A teg <u> </u> : Deixa qualquer frase ou palavra sublinhada.
  13. A teg <sup> </sup>define texto sobrescrito. texto sobrescrito parece metade de um caráter acima da linha normal, e às vezes é processado em uma fonte menor. Superscript texto pode ser utilizado para notas de rodapé, ou numeros 22.

Existem varias tegs e atributos, acesse o site w3bai.com ou w3schools.com/ nele vai mostra varias tegs de html, css, javascrip entre outras.
